# dotfiles

Cross-platform Zsh config with Oh My Zsh, works on macOS / Linux / WSL.

## Install

```sh
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)" && git clone git@github.com:nothingness0db/dotfiles.git ~/dotfiles && cp ~/dotfiles/.zshrc ~/.zshrc && source ~/.zshrc
```
